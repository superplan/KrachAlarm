Hintergrundbild & App-Icon
==========================

Lege dein Foto hier ab und nenne es genau:

    foto.png

Dann wird es automatisch verwendet als:
  - Hintergrund der App (verschwommen + heller Schleier, damit alles lesbar bleibt)
  - App-Icon der PWA

Empfehlung:
  - möglichst quadratisch (z. B. 512 x 512 Pixel) für ein sauberes Icon
  - Format PNG (Dateiname muss "foto.png" sein)

Hinweis: Für das Icon der echten Android-APK werden zusätzlich eigene
Icon-Dateien gebraucht – die erzeugt Claude aus deinem foto.png, sobald
die Datei hier liegt.
